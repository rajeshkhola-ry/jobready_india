# V1-V2-V3 Merge Policy (Owner Approved)

Date: 2026-07-14

1. V1 home page remains the baseline and should continue as-is.
2. V2 and future V3 work must not break or replace V1 home behavior.
3. During merge, only minor UI improvements are allowed on V1 (spacing/visual polish), no flow change.
4. Repeating tools or duplicate actions are not allowed in merged home experience.
5. Prefer one clear access path per feature/tool.
6. Optimize page space and keep the app clean and fast on web/mobile.

Implementation note:
- Repeated V2 benchmark button inside tool grid has been removed; top navigation benchmark access remains.
